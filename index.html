什么是JT？

为web开发者提供开发javascript辅助开发工具及半自动化工具。

  • 代理服务
  • 项目构建
  • 插件支持

快速入门

通过npm快速安装

npm install -g jt-cli

注意:请一定加上-g参数

定位到项目文件夹根目录，执行下面命令生成配置文件

jt init

命令执行后会在当前路径下生成.jt文件夹

├──┬.jt
   ├────config.js
   ├────fs.js
   ├────project.js
   └────server.js
